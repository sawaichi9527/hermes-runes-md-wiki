# Workspace verification history

This directory stores historical milestone verification notes that were moved out of the runtime workspace seed.

These files are development and release evidence, not default trusted workspace memory.

## Boundary

- `wiki/<workspace slug>/` should hold curated workspace memory files.
- `wiki/<workspace slug>/forge-inbox/` should hold draft / untrusted proposals.
- Historical milestone notes such as `verification-m*.md` belong under `dev/wiki-history/<workspace slug>/verification/`.
- These notes may be used for developer review and release archaeology, but they should not be treated as default workspace seed files.

## Origin
