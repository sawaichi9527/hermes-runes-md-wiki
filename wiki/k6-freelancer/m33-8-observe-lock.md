# Four-State Invocation Boundary (Frozen)

The following state machine is now governance-locked.

| State         | Definition                                                             | Action                       |
| ------------- | ---------------------------------------------------------------------- | ---------------------------- |
| MATCH         | Explicit core incantation + activation + Hermes Runes context          | invoke_bundle=true           |
| CONFIRM       | Ritual traits detected but intent/context unclear                      | ritual confirmation required |
| CONFIRM_MATCH | User affirmation after pending confirmation                            | invoke_bundle=true           |
| NO_MATCH      | Mythology / fiction / gaming / diagnostics / unrelated ritual language | invoke_bundle=false          |
