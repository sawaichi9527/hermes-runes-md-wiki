# Long-term Objectives Index

Status: P0 baseline

## Purpose

This index tracks long-running project/domain objective namespaces.

Objective namespaces are intended for:

- engineering projects
- long-running domains
- platform baselines
- operational lifecycle tracking
- complex relationship groups

## Objective Namespace Rule

Every objective namespace should contain:

```text
wiki/<objective-slug>/README.md
```

The objective README acts as the local relationship guide for that namespace.

## Promotion Guidance

Flat-first files may later be promoted into objective namespaces if:

- file count grows
- lifecycle complexity grows
- verification/operations history matters
- relationship management becomes difficult
