# Public Beta Safety Policy

Hermes Runes MD Wiki is prepared for public Open Beta evaluation.

## Scope

```text
branch: main
phase: Open Beta
support: best effort
intended use: personal/local evaluation
```

## Local Data Rule

Do not commit local private data, local credentials, `.env` files, private keys, raw confidential prompts, raw confidential outputs, local logs, or local database files.

Use local-only configuration files and keep runtime artifacts outside git.

## Memory Review Rule

Usage evidence, observations, beta feedback, and proposal drafts are not trusted memory by default.

Review them before promoting anything into governed Markdown memory.

## Reporting

For sensitive issues, contact the repository owner privately before opening a public issue.

For non-sensitive Open Beta feedback, use the normal finding intake flow.
