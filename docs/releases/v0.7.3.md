## Summary

```text
single main agent
```

## Local context model

M223 assumes local finite compute:

```text
maximum context: 128K tokens
compression pressure point: about 50 percent context usage
```

The intended division is:

```text
conversation = volatile working memory
Kanban = compact task-state checkpoint
Runes Wiki = governed durable source-of-truth
Git history = exact change evidence
```

Kanban stores state, next action, blocker, and evidence pointers. It is not a workflow engine.
