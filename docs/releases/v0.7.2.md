# Hermes Runes MD Wiki v0.7.2

Release date: 2026-06-14

## Summary

v0.7.2 is a small safety and documentation release for existing local installations.

The main addition is `runes-wiki-migration-guard`, a minimal local helper that backs up `wiki/`, checks incoming repository updates, and stops before pull if an update may touch possible user-owned Markdown.

This release does not expand Runes Shield, does not add a daemon, does not add Git hooks, and does not introduce an enterprise migration framework.

## Highlights

- Added `bin/runes-wiki-migration-guard`.
- Added `tools/wiki_migration_guard/migration_guard.py`.
- Added local `wiki/` backup before guarded updates.
- Added incoming update risk classification:
  - SAFE for non-wiki changes
  - CAUTION for system/index wiki changes
  - STOP for possible user-owned Markdown changes
- Added same-second backup suffix handling.
- Documented existing-installation update flow in README.
- Verified real safe-path dogfood with `./bin/runes-wiki-migration-guard update`.

## Existing installation update path

Existing local installations should prefer:

```bash
./bin/runes-wiki-migration-guard update
```

instead of a naked `git pull` when `wiki/` may contain personal or project knowledge.

## Scope boundary

v0.7.2 does not introduce:

- migration guard auto-restore
- migration guard repair/apply behavior
- automatic user-owned Markdown merge
- Git hook installation
- daemon behavior
- Runes Shield schema changes
- proposal governance changes
- OPC runtime changes
- enterprise-grade migration management
