# Hermes Runes MD Wiki v0.7.1

Release date: 2026-06-14

## Summary

v0.7.1 is a small compatibility and documentation release for the optional Hermes Agent OPC workspace overlay.

This release preserves the existing single-agent governed Runes Shield flow and keeps OPC support as an optional workspace organization pattern. It does not add an OPC runtime, scheduler, dispatcher, enterprise memory layer, or a new Runes Shield schema.

## Highlights

- Added optional OPC workspace overlay documentation.
- Added optional `wiki/<workspace>/opc/` profile memory seed layout.
- Added role memory guidance for `secretary`, `coordinator`, `researcher`, `writer`, `builder`, and `runes-holder`.
- Documented that OPC Runes access remains secretary-consent-gated and runes-holder/Shield-gated.
- Linked the optional OPC overlay from README and system policy index.
- Preserved the default single-agent workspace seed and existing Runes Shield contract.

## Scope boundary

v0.7.1 does not introduce:

- a new runtime mode
- parallel profile dispatch
- direct profile-to-wiki access
- a second Runes Shield request format
- automatic proposal promotion
- enterprise-grade orchestration or memory management
