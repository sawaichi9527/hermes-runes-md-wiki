# v0.5.0 Tester Checklist

Status: public tester checklist / released baseline runtime surface review  
Baseline: `v0.5.0`  
Current development target: `v0.7.0-dev`  
Date: 2026-06-08

## Purpose

Use this checklist to confirm a fresh public tester checkout is using the intended released `v0.5.0` runtime surface.

v0.5.0 supersedes v0.3.0 as the recommended released Open Beta baseline.

For a fresh install from current `main` / `v0.7.0-dev`, use:

```text
docs/fresh-install-manual.md
```

## Checkout verification

For the released `v0.5.0` baseline, run:

```bash
git fetch --tags
git checkout v0.5.0
cat VERSION
```

Expected version:

```text
0.5.0
```

For current development-line fresh install validation, run from `main` instead and expect:

```text
0.7.0-dev
```

Expected tags:

```bash
git tag --list --sort=-creatordate | sed -n '1,10p'
```

Expected result includes:

```text
v0.5.0
v0.3.0
v0.1.0-beta.1
```

## Runtime smoke gate

Run from repository root for the released `v0.5.0` baseline:

```bash
./bin/hermes-runes decipher freshness --json
./bin/hermes-runes probe policy --json
./bin/hermes-backend-check
./bin/hermes-memory-migrate
./bin/hermes-memory-security-scan
./bin/hermes-recall "forge inbox boundary" --project freelancer --mode hybrid --limit 5 --json
./bin/hermes-memory-smoke
./bin/hermes-retrieval-governance-smoke
```

For current `main` / `v0.7.0-dev` core-profile fresh install validation, follow the FTS-first manual path in `docs/fresh-install-manual.md`.

## Runtime surface expectation

The v0.5.0 runtime surface should keep:

- user-facing CLI wrappers under `bin/`
- runtime/support tools under root `tools/`
- development-only smoke, fixtures, regression, trial, and historical assets under `dev/`

Root `tools/runes_shield` should not contain historical smoke or fixture directories.

## Boundary

v0.5.0 is suitable for personal/local Open Beta evaluation.

It is not:

- a production memory service
- an enterprise support commitment
- a multi-user SaaS platform
- an autonomous trusted memory writer
