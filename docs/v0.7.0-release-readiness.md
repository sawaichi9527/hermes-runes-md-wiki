# v0.7.0 Release Readiness

Status: PREP / release gate pending  
Version line: 0.7.0-dev  
Date: 2026-06-08

## Scope

v0.7.0 is the fresh-install hardening and Open Beta baseline advancement line for Hermes Runes MD Wiki.

This release line promotes the public tester target beyond the earlier v0.5.0 runtime baseline by adding a validated fresh-install core path, PostgreSQL / pgvector setup guidance, repo and venv reset separation, and fresh-install tooling defaults that no longer require runtime overrides.

## Release boundary

v0.7.0 remains personal/local Open Beta scope.

It is intended for:

- fresh clone evaluation
- local PostgreSQL / pgvector setup validation
- Markdown wiki import validation
- core FTS retrieval smoke validation
- governed memory boundary review
- public tester onboarding improvement

It is not:

- a stable production release
- an enterprise support commitment
- a hosted service
- an autonomous trusted memory writer
- a multi-user SaaS platform

## Fresh-install hardening included

v0.7.0 includes the M204 / M205 fresh-install hardening path:

- standalone fresh-install manual

- separate PostgreSQL stack/data removal
- separate repository and local venv removal

- local port 5433 preflight
- PostgreSQL / pgvector stack initialization
